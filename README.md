# SimpleWarikanApp

これは、簡単な割り勘計算を行うためのAndroidアプリケーションです。

## 概要

飲み会やランチなど、複数人での支払い場面で、合計金額と人数を入力するだけで、一人当たりの金額を素早く計算できます。

## 機能

*   合計金額の入力
*   人数の入力
*   一人当たりの金額の自動計算と表示

## 使い方

1.  「金額」の入力フィールドに、支払う合計金額を入力します。
2.  「人数」の入力フィールドに、割り勘する人数を入力します。
3.  入力が完了すると、自動的に一人当たりの金額が計算され、表示されます。

## 開発環境

*   言語: Kotlin
*   UI: Jetpack Compose
